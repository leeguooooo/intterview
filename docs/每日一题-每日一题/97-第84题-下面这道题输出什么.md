# 第84题 下面这道题输出什么
```js
    var a = {n:1}
    var b = a
    a.x = a = {n:2}
    
    console.log(a)
    console.log(b)
    a.n = 3
    console.log(b)
```**答案**  
```js
    // a {n:2}
    // b {n:1,x:{n:2}}
    // b {n:1,x:{n:3}}
```

**解析**

  * `a.x`的`.`比赋值`=`的优先级高
  * `a.x = 100`可拆解为 
    * `a.x = undefined` 初始化`a.x`的属性
    * `a.x = 100` 为`x`赋值
