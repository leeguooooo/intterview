# 第40题 实现 (5).add(3).minus(2) 功能

> 例： 5 + 3 - 2，结果为 6
```js
    (function() {
      function add(val) {
        if(typeof val !== 'number' || Number.isNaN(val)) throw TypeError('请输入数字')
        return this + val
      }
      
      function minus(val) {
        if(typeof val !== 'number' || Number.isNaN(val)) throw TypeError('请输入数字')
        return this - val
      }
      
      Number.prototype.add = add
      Number.prototype.minus = minus
    })()
```
