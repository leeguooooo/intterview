# 3 层叠上下文

![层叠上下文与 z-index:position/opacity/transform 等触发新上下文,同一上下文内 z-index 才可比,决定 z 轴层叠顺序](/images/diagrams/css-stacking-context.webp)

> 元素提升为一个比较特殊的图层，在三维空间中 (z轴) 高出普通元素一等。

**触发条件**

  * 根层叠上下文(`html`)
  * `position`
  * `css3`属性 
    * `flex`
    * `transform`
    * `opacity`
    * `filter`
    * `will-change`
    * `webkit-overflow-scrolling`

**层叠等级：层叠上下文在z轴上的排序**

  * 在同一层叠上下文中，层叠等级才有意义
  * `z-index`的优先级最高
