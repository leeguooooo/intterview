### Flex实现两列布局

> 这里简单实现一下（其实应该使用flex-basis属性）：
```html
    <div class="box">
      <div class="box-left"></div>
      <div class="box-right"></div>
    </div>
```  
```css
    .box {
      height: 200px;
      display: flex;
    }
    
    .box > div {
      height: 100%;
    }
    
    .box-left {
      width: 200px;
      background-color: blue;
    }
    
    .box-right {
      flex: 1; // 设置flex-grow属性为1，默认为0
      background-color: red;
    }
```
