  • йЎөйқўеҜјиҲӘпјҡз”ЁжҲ·иҫ“е…Ҙ URLпјҢжөҸи§ҲеҷЁиҝӣзЁӢиҝӣиЎҢиҜ·жұӮе’ҢеҮҶеӨҮеӨ„зҗҶгҖӮ
  • йЎөйқўжёІжҹ“пјҡиҺ·еҸ–еҲ°зӣёе…іиө„жәҗеҗҺпјҢжёІжҹ“еҷЁиҝӣзЁӢиҙҹиҙЈйҖүйЎ№еҚЎеҶ…йғЁзҡ„жёІжҹ“еӨ„зҗҶгҖӮ

1. йЎөйқўеҜјиҲӘиҝҮзЁӢ

еҪ“з”ЁжҲ·еңЁең°еқҖж Ҹдёӯиҫ“е…ҘеҶ…е®№ж—¶пјҢжөҸи§ҲеҷЁеҶ…йғЁдјҡиҝӣиЎҢд»ҘдёӢеӨ„зҗҶгҖӮ

  • йҰ–е…ҲжөҸи§ҲеҷЁиҝӣзЁӢзҡ„ UI зәҝзЁӢдјҡиҝӣиЎҢеӨ„зҗҶпјҡеҰӮжһңжҳҜ URIпјҢеҲҷдјҡеҸ‘иө·зҪ‘з»ңиҜ·жұӮжқҘиҺ·еҸ–зҪ‘з«ҷеҶ…е®№пјӣеҰӮжһңдёҚжҳҜпјҢеҲҷиҝӣе…Ҙжҗңзҙўеј•ж“ҺгҖӮ
  • еҰӮжһңйңҖиҰҒеҸ‘иө·зҪ‘з»ңиҜ·жұӮпјҢиҜ·жұӮиҝҮзЁӢз”ұзҪ‘з»ңзәҝзЁӢжқҘе®ҢжҲҗгҖӮHTTP иҜ·жұӮе“Қеә”еҰӮжһңжҳҜ HTML ж–Үд»¶пјҢеҲҷе°Ҷж•°жҚ®дј йҖ’еҲ°жёІжҹ“еҷЁиҝӣзЁӢпјӣеҰӮжһңжҳҜе…¶д»–ж–Үд»¶еҲҷж„Ҹе‘ізқҖиҝҷжҳҜдёӢиҪҪиҜ·жұӮпјҢжӯӨж—¶дјҡе°Ҷж•°жҚ®дј йҖ’еҲ°дёӢиҪҪз®ЎзҗҶеҷЁгҖӮ
  • еҰӮжһңиҜ·жұӮе“Қеә”дёә HTML еҶ…е®№пјҢжӯӨж—¶жөҸи§ҲеҷЁеә”еҜјиҲӘеҲ°иҜ·жұӮз«ҷзӮ№пјҢзҪ‘з»ңзәҝзЁӢдҫҝйҖҡзҹҘ UI зәҝзЁӢж•°жҚ®еҮҶеӨҮе°ұз»ӘгҖӮ
  • жҺҘдёӢжқҘпјҢUI зәҝзЁӢдјҡеҜ»жүҫдёҖдёӘжёІжҹ“еҷЁиҝӣзЁӢжқҘиҝӣиЎҢзҪ‘йЎөжёІжҹ“гҖӮеҪ“ж•°жҚ®е’ҢжёІжҹ“еҷЁиҝӣзЁӢйғҪеҮҶеӨҮеҘҪеҗҺпјҢHTML ж•°жҚ®йҖҡиҝҮ IPC д»ҺжөҸи§ҲеҷЁиҝӣзЁӢдј йҖ’еҲ°жёІжҹ“еҷЁиҝӣзЁӢдёӯгҖӮ
  • жёІжҹ“еҷЁиҝӣзЁӢжҺҘж”¶ HTML ж•°жҚ®еҗҺпјҢе°ҶејҖе§ӢеҠ иҪҪиө„жәҗе№¶жёІжҹ“йЎөйқўгҖӮ
  • жёІжҹ“еҷЁиҝӣзЁӢе®ҢжҲҗжёІжҹ“еҗҺпјҢйҖҡиҝҮ IPC йҖҡзҹҘжөҸи§ҲеҷЁиҝӣзЁӢйЎөйқўе·ІеҠ иҪҪгҖӮ

д»ҘдёҠжҳҜз”ЁжҲ·еңЁең°еқҖж Ҹиҫ“е…ҘзҪ‘з«ҷең°еқҖпјҢеҲ°йЎөйқўејҖе§ӢжёІжҹ“зҡ„ж•ҙдҪ“иҝҮзЁӢгҖӮдёәдәҶж–№дҫҝзҗҶи§ЈпјҢжҲ‘её®дҪ жўізҗҶдәҶдёҖдёӘжөҒзЁӢеӣҫпјҡ

еҰӮжһңеҪ“еүҚйЎөйқўи·іиҪ¬еҲ°е…¶д»–зҪ‘з«ҷпјҢжөҸи§ҲеҷЁе°Ҷи°ғз”ЁдёҖдёӘеҚ•зӢ¬зҡ„жёІжҹ“иҝӣзЁӢжқҘеӨ„зҗҶж–°еҜјиҲӘпјҢеҗҢж—¶дҝқз•ҷеҪ“еүҚжёІжҹ“иҝӣзЁӢжқҘеӨ„зҗҶеғҸunloadиҝҷзұ»дәӢд»¶гҖӮ

еңЁдёҠйқўзҡ„иҝҮзЁӢдёӯеҸҜд»ҘзңӢеҲ°пјҢйЎөйқўеҜјиҲӘдё»иҰҒдҫқиө–жөҸи§ҲеҷЁиҝӣзЁӢгҖӮе…¶дёӯпјҢдёҠиҝ°иҝҮзЁӢдёӯзҡ„жӯҘйӘӨ 5 дҫҝжҳҜйЎөйқўзҡ„жёІжҹ“йғЁеҲҶпјҢиҜҘиҝҮзЁӢеҗҢж ·дҫқиө–жёІжҹ“еҷЁиҝӣзЁӢпјҢжҲ‘д»¬дёҖиө·жқҘзңӢзңӢ

2. йЎөйқўжёІжҹ“иҝҮзЁӢ

еүҚйқўиҜҙиҝҮпјҢжёІжҹ“еҷЁиҝӣзЁӢиҙҹиҙЈйҖүйЎ№еҚЎеҶ…йғЁеҸ‘з”ҹзҡ„жүҖжңүдәӢжғ…пјҢе®ғзҡ„ж ёеҝғе·ҘдҪңжҳҜе°Ҷ HTMLгҖҒCSS е’Ң JavaScript иҪ¬жҚўдёәеҸҜдәӨдә’зҡ„йЎөйқўгҖӮ

ж•ҙдҪ“дёҠпјҢжёІжҹ“еҷЁиҝӣзЁӢжёІжҹ“йЎөйқўзҡ„жөҒзЁӢеҹәжң¬еҰӮдёӢгҖӮ

  • и§Јжһҗ(Parser)пјҡи§Јжһҗ HTML/CSS/JavaScript д»Јз ҒгҖӮ
  • еёғеұҖ(Layout)пјҡе®ҡдҪҚеқҗж Үе’ҢеӨ§е°ҸгҖҒжҳҜеҗҰжҚўиЎҢгҖҒеҗ„з§Қposition/overflow/z-indexеұһжҖ§зӯүи®Ўз®—гҖӮ
  • з»ҳеҲ¶(Paint)пјҡеҲӨж–ӯе…ғзҙ жёІжҹ“еұӮзә§йЎәеәҸгҖӮ
  • е…үж …еҢ–(Raster)пјҡе°Ҷи®Ўз®—еҗҺзҡ„дҝЎжҒҜиҪ¬жҚўдёәеұҸе№•дёҠзҡ„еғҸзҙ гҖӮ

1. и§Јжһҗ

жёІжҹ“еҷЁиҝӣзЁӢзҡ„дё»зәҝзЁӢдјҡи§Јжһҗд»ҘдёӢеҶ…е®№пјҡ

  • и§Јжһҗ HTML еҶ…е®№пјҢдә§з”ҹдёҖдёӘ DOM иҠӮзӮ№ж ‘пјӣ
  • и§Јжһҗ CSSпјҢдә§з”ҹ CSS и§„еҲҷж ‘пјӣ
  • и§Јжһҗ Javascript и„ҡжң¬пјҢз”ұдәҺ Javascript и„ҡжң¬еҸҜд»ҘйҖҡиҝҮ DOM API е’Ң CSSOM API жқҘж“ҚдҪң DOM иҠӮзӮ№ж ‘е’Ң CSS и§„еҲҷж ‘пјҢеӣ жӯӨиҜҘиҝҮзЁӢдёӯдјҡзӯүеҫ… JavaScript иҝҗиЎҢе®ҢжҲҗжүҚз»§з»ӯи§Јжһҗ HTML

и§Јжһҗе®ҢжҲҗеҗҺпјҢжҲ‘д»¬еҫ—еҲ°дәҶ DOM иҠӮзӮ№ж ‘е’Ң CSS и§„еҲҷж ‘пјҢеёғеұҖиҝҮзЁӢдҫҝжҳҜйҖҡиҝҮ DOM иҠӮзӮ№ж ‘е’Ң CSS и§„еҲҷж ‘жқҘжһ„йҖ жёІжҹ“ж ‘пјҲRender Treeпјү

2. еёғеұҖ

йҖҡиҝҮи§Јжһҗд№ӢеҗҺпјҢжёІжҹ“еҷЁиҝӣзЁӢзҹҘйҒ“жҜҸдёӘиҠӮзӮ№зҡ„з»“жһ„е’Ңж ·ејҸпјҢдҪҶеҰӮжһңйңҖиҰҒжёІжҹ“йЎөйқўпјҢжөҸи§ҲеҷЁиҝҳйңҖиҰҒиҝӣиЎҢеёғеұҖпјҢеёғеұҖиҝҮзЁӢдҫҝжҳҜжҲ‘д»¬еёёиҜҙзҡ„жёІжҹ“ж ‘зҡ„еҲӣе»әиҝҮзЁӢгҖӮ

еңЁиҝҷдёӘиҝҮзЁӢдёӯпјҢеғҸheaderжҲ–display:noneзҡ„е…ғзҙ пјҢе®ғд»¬дјҡеӯҳеңЁ DOM иҠӮзӮ№ж ‘дёӯпјҢдҪҶдёҚдјҡиў«ж·»еҠ еҲ°жёІжҹ“ж ‘йҮҢгҖӮ

еёғеұҖе®ҢжҲҗеҗҺпјҢе°Ҷдјҡиҝӣе…Ҙз»ҳеҲ¶зҺҜиҠӮ

3. з»ҳеҲ¶

еңЁз»ҳеҲ¶жӯҘйӘӨдёӯпјҢжёІжҹ“еҷЁдё»зәҝзЁӢдјҡйҒҚеҺҶжёІжҹ“ж ‘жқҘеҲӣе»әз»ҳеҲ¶и®°еҪ•гҖӮ

йңҖиҰҒжіЁж„Ҹзҡ„жҳҜпјҢеҰӮжһңжёІжҹ“ж ‘еҸ‘з”ҹдәҶж”№еҸҳпјҢеҲҷжёІжҹ“еҷЁдјҡи§ҰеҸ‘йҮҚз»ҳпјҲRepaintпјүе’ҢйҮҚжҺ’пјҲReflowпјүгҖӮ

  • йҮҚз»ҳ пјҡеұҸе№•зҡ„дёҖйғЁеҲҶиҰҒйҮҚз”»пјҢжҜ”еҰӮжҹҗдёӘ CSS зҡ„иғҢжҷҜиүІеҸҳдәҶпјҢдҪҶжҳҜе…ғзҙ зҡ„еҮ дҪ•е°әеҜёжІЎжңүеҸҳгҖӮ
  • йҮҚжҺ’ пјҡе…ғзҙ зҡ„еҮ дҪ•е°әеҜёеҸҳдәҶпјҲжёІжҹ“ж ‘зҡ„дёҖйғЁеҲҶжҲ–е…ЁйғЁеҸ‘з”ҹдәҶеҸҳеҢ–пјүпјҢйңҖиҰҒйҮҚж–°йӘҢиҜҒе№¶и®Ўз®—жёІжҹ“ж ‘гҖӮ

дёәдәҶдёҚеҜ№жҜҸдёӘе°Ҹзҡ„еҸҳеҢ–йғҪиҝӣиЎҢе®Ңж•ҙзҡ„еёғеұҖи®Ўз®—пјҢжёІжҹ“еҷЁдјҡе°Ҷжӣҙж”№зҡ„е…ғзҙ е’Ңе®ғзҡ„еӯҗе…ғзҙ иҝӣиЎҢи„ҸдҪҚж Үи®°пјҢиЎЁзӨәиҜҘе…ғзҙ йңҖиҰҒйҮҚж–°еёғеұҖгҖӮе…¶дёӯпјҢе…ЁеұҖж ·ејҸжӣҙж”№дјҡи§ҰеҸ‘е…ЁеұҖеёғеұҖпјҢйғЁеҲҶж ·ејҸжҲ–е…ғзҙ жӣҙж”№дјҡи§ҰеҸ‘еўһйҮҸеёғеұҖпјҢеўһйҮҸеёғеұҖжҳҜејӮжӯҘе®ҢжҲҗзҡ„пјҢе…ЁеұҖеёғеұҖеҲҷдјҡеҗҢжӯҘи§ҰеҸ‘гҖӮ

йҮҚжҺ’йңҖиҰҒж¶үеҸҠеҸҳжӣҙзҡ„жүҖжңүзҡ„з»“зӮ№еҮ дҪ•е°әеҜёе’ҢдҪҚзҪ®пјҢжҲҗжң¬жҜ”йҮҚз»ҳзҡ„жҲҗжң¬й«ҳеҫ—еӨҡзҡ„еӨҡгҖӮжүҖд»ҘжҲ‘д»¬иҰҒжіЁж„Ҹд»ҘйҒҝе…Қйў‘з№Ғең°иҝӣиЎҢеўһеҠ гҖҒеҲ йҷӨгҖҒдҝ®ж”№ DOM з»“зӮ№гҖҒз§»еҠЁ DOM зҡ„дҪҚзҪ®гҖҒResize зӘ—еҸЈгҖҒж»ҡеҠЁзӯүж“ҚдҪңпјҢеӣ дёәиҝҷдәӣж“ҚдҪңеҸҜиғҪдјҡеҜјиҮҙжҖ§иғҪйҷҚдҪҺгҖӮ

4. е…үж …еҢ–

йҖҡиҝҮи§ЈжһҗгҖҒеёғеұҖе’Ңз»ҳеҲ¶иҝҮзЁӢпјҢжөҸи§ҲеҷЁиҺ·еҫ—дәҶж–ҮжЎЈзҡ„з»“жһ„гҖҒжҜҸдёӘе…ғзҙ зҡ„ж ·ејҸгҖҒз»ҳеҲ¶йЎәеәҸзӯүдҝЎжҒҜгҖӮе°ҶиҝҷдәӣдҝЎжҒҜиҪ¬жҚўдёәеұҸе№•дёҠзҡ„еғҸзҙ пјҢиҝҷдёӘиҝҮзЁӢиў«з§°дёәе…үж …еҢ–

е…үж …еҢ–еҸҜд»Ҙиў« GPU еҠ йҖҹпјҢе…үж …еҢ–еҗҺзҡ„дҪҚеӣҫдјҡиў«еӯҳеӮЁеңЁ GPU еҶ…еӯҳдёӯгҖӮж №жҚ®еүҚйқўд»Ӣз»Қзҡ„жёІжҹ“жөҒзЁӢпјҢеҪ“йЎөйқўеёғеұҖеҸҳжӣҙдәҶдјҡи§ҰеҸ‘йҮҚжҺ’е’ҢйҮҚз»ҳпјҢиҝҳйңҖиҰҒйҮҚж–°иҝӣиЎҢе…үж …еҢ–гҖӮжӯӨж—¶еҰӮжһңйЎөйқўдёӯжңүеҠЁз”»пјҢеҲҷдё»зәҝзЁӢдёӯиҝҮеӨҡзҡ„и®Ўз®—д»»еҠЎеҫҲеҸҜиғҪдјҡеҪұе“ҚеҠЁз”»зҡ„жҖ§иғҪ

еӣ жӯӨпјҢзҺ°д»Јзҡ„жөҸи§ҲеҷЁйҖҡеёёдҪҝз”ЁеҗҲжҲҗзҡ„ж–№ејҸпјҢе°ҶйЎөйқўзҡ„еҗ„дёӘйғЁеҲҶеҲҶжҲҗиӢҘе№ІеұӮпјҢеҲҶеҲ«еҜ№е…¶иҝӣиЎҢж …ж јеҢ–пјҲе°Ҷе®ғд»¬еҲҶеүІжҲҗдәҶдёҚеҗҢзҡ„з“ҰзүҮпјүпјҢе№¶йҖҡиҝҮеҗҲжҲҗеҷЁзәҝзЁӢиҝӣиЎҢйЎөйқўзҡ„еҗҲжҲҗ

еҗҲжҲҗиҝҮзЁӢеҰӮдёӢпјҡ

  • еҪ“дё»зәҝзЁӢеҲӣе»әдәҶеҗҲжҲҗеұӮе№¶зЎ®е®ҡдәҶз»ҳеҲ¶йЎәеәҸпјҢдҫҝе°ҶиҝҷдәӣдҝЎжҒҜжҸҗдәӨз»ҷеҗҲжҲҗзәҝзЁӢпјӣ
  • еҗҲжҲҗеҷЁзәҝзЁӢе°ҶжҜҸдёӘеӣҫеұӮж …ж јеҢ–пјҢз„¶еҗҺе°ҶжҜҸдёӘеӣҫеқ—еҸ‘йҖҒз»ҷе…үж …зәҝзЁӢ
  • е…үж …зәҝзЁӢж …ж јеҢ–жҜҸдёӘз“ҰзүҮпјҢе№¶е°Ҷе®ғд»¬еӯҳеӮЁеңЁ GPU еҶ…еӯҳдёӯ
  • еҗҲжҲҗеҷЁзәҝзЁӢйҖҡиҝҮ IPC жҸҗдәӨз»ҷжөҸи§ҲеҷЁиҝӣзЁӢпјҢиҝҷдәӣеҗҲжҲҗеҷЁеё§иў«еҸ‘йҖҒеҲ° GPU иҝӣзЁӢеӨ„зҗҶпјҢе№¶жҳҫзӨәеңЁеұҸе№•дёҠгҖӮ

еҗҲжҲҗзҡ„зңҹжӯЈзӣ®зҡ„жҳҜпјҢеңЁз§»еҠЁеҗҲжҲҗеұӮзҡ„ж—¶еҖҷдёҚз”ЁйҮҚж–°е…үж …еҢ–гҖӮеӣ дёәжңүдәҶеҗҲжҲҗеҷЁзәҝзЁӢпјҢйЎөйқўжүҚеҸҜд»ҘзӢ¬з«ӢдәҺдё»зәҝзЁӢиҝӣиЎҢжөҒз•…зҡ„ж»ҡеҠЁ
